To install required packages set up a virtual environment and use command
   "pip install -r requirements.txt"

Also saw this as a way to do it in one line 
    "virtualenv .env && source .env/bin/activate && pip install -r requirements.txt"
    Found on stackOverflow - https://stackoverflow.com/questions/6590688/is-it-bad-to-have-my-virtualenv-directory-inside-my-git-repository
